assuming you have the appropriate AWS access key and secret key set:

install dependencies

```bash
npm install -g serverless
npm install
```

deploy api and populate users table

```bash
serverless deploy -v
node api/users/populateUsers.js
```

teardown api

```bash
serverless remove -v
```
